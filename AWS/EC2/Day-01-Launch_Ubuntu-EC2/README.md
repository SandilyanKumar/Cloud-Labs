# AWS EC2 - Day 01: Launch an EC2 Instance

## Objective

Launch an Ubuntu EC2 instance using the AWS Management Console.

---

## Prerequisites

- AWS Account
- IAM User
- EC2 Console Access

---

## Services Used

- Amazon EC2
- Security Group
- Key Pair

---

## Steps

1. Login to AWS Console.
2. Open EC2 Dashboard.
3. Click Launch Instance.
4. Select Ubuntu 24.04 LTS.
5. Choose t2.micro.
6. Create Key Pair.
7. Configure Security Group.
8. Launch the instance.

---

## Verification

- Instance state is Running.
- SSH connection is successful.

---

## Result

Successfully launched an EC2 instance.

---

## Learning Outcome

- Learned how to launch an EC2 instance.
- Understood AMI, Instance Type, Key Pair, and Security Group.
