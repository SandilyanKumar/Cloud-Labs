# Amazon EC2 (Elastic Compute Cloud)

## 📖 Overview

Amazon Elastic Compute Cloud (Amazon EC2) is an AWS compute service that enables you to create, manage, and scale virtual servers in the cloud. It provides secure, resizable compute capacity and supports Linux and Windows operating systems for a wide range of workloads.

# 📌 Best Practices

- Follow the AWS Well-Architected Framework.
- Use IAM Roles instead of storing credentials.
- Allow only required ports in Security Groups.
- Create regular AMI and EBS Snapshots.
- Monitor instances using Amazon CloudWatch.
- Use Tags for resource management.
- Stop unused instances to reduce cost.
